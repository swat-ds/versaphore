Using Versaphore

Note that this is experimental software and may change during the course of the semester. If you have any questions or comments, please contact digitalscholarship@swarthmore.edu.

Step 1

Open the Google Sheets template and make a copy for your own drive.

Step 2

Break the translations of your poems into segments according to the template example. /br indicates a linebreak and two /br fields in succession will create stanza break. Align parts of the text you’d like to compare using the /app tag in the frst column.

Step 3

From your Google Sheets file, choose the File menu and select Publish to the web. From here, click Publish. A URL should appear that you’ll need to copy – CTL + C (pc) or CTL + ⌘ (mac).

Step 4

Add a workspace using the Workspace menu to the right of the screen.

  • A workspace name should not contain punctuation or spaces
  • In the Google Sheet URL field, enter the entire URL from the “publish to the web menu”

Exploring the Workspace

Choose the workspace from the Workspace menu. In the Witness menu, you can choose which base translation appears. Using the Ignore List, you can select which translations to ignore when calculating the difference between segments.
